# Instalar e configurar uma máquina Ubuntu Server no VirtualBox



## Objetivo



Criar uma máquina Ubuntu Server no VirtualBox e habilitar acesso SSH.



## Requisitos



- VirtualBox instalado

- ISO Ubuntu Server

- 4GB RAM recomendados

- 25GB de disco



## Criando a máquina virtual



1. Abrir VirtualBox

2. Clicar em Novo

3. Nome: Ubuntu Server

4. Selecionar ISO

5. Configurar memória

6. Criar disco
